# Links Component (Footer)

## Описание
Компонент футера с социальными сетями, реквизитами и отслеживанием заказа.

## Путь
`components/app/footer/Links.vue`

## Props
Нет (использует данные конфигурации/локализации)

## Состояния (State)
Нет (без состояния, статический компонент)

## События (Events)
- `trackingButton` - клик по ссылке отслеживания заказа
- `telegramButton` - клик по ссылке Telegram
- `vkButton` - клик по ссылке VK
- `requisitesButton` - клик по ссылке реквизитов

## Функционал
Компонент отображает социальные сети и полезные ссылки в футере страницы.

### Основные функции:
1. **Социальные сети:**
   - Telegram (@kalashyulyaa)
   - VK (@kalashyulya)
   - Иконки с hover эффектами
   - Локализованные названия через `printLocale`

2. **Реквизиты:**
   - Ссылка на `/requisites`
   - Локализованный текст
   - Иконка

3. **Отслеживание заказа** (новое):
   - Ссылка на `/shop/tracking`
   - Текст: "Отслеживание заказа"
   - Подтекст: "Найти заказ по ID платежа"
   - Иконка поиска/документа
   - Метрики отслеживания клика

4. **Локализация:**
   - Использование `useLocales()` для получения переведенных текстов
   - Все названия и описания через `printLocale`

5. **Метрики:**
   - Отслеживание всех кликов через `metrics.trackButtonClick`

## UI/UX
- Вертикальная ориентация элементов
- Группировка элементов
- Hover эффекты с изменением цвета фона
- Адаптивность: на мобильных устройствах элементы по центру, на десктопе слева
- Иконки для визуального определения

## Зависимости (Dependencies)
- `composables/useLocales` - для локализации текстов
- `utils/metrics` - для метрик кликов
- SVG иконки из `public/` директории

## Интеграция
Интеграция с Nuxt через стандартные компоненты и composables.

## Примеры использования

```vue
<!-- Social Networks -->
<a href="https://t.me/kalashyulyaa" class="...">
  <img src="/links/telegram.svg" alt="Telegram" />
  <div class="...">
    <span>Telegram</span>
    <span>@kalashyulyaa</span>
  </div>
</a>

<!-- Order Tracking (NEW) -->
<NuxtLink to="/shop/tracking" class="...">
  <div class="...">
    <span>Отслеживание заказа</span>
    <span>Найти заказ по ID платежа</span>
  </div>
</NuxtLink>
```

## Локализация
Использует следующие ключи локализации:
- `footer_telegram_title` - заголовок Telegram
- `footer_vk_title` - заголовок VK
- `footer_linkname` - общий подзаголовок ссылок
- `footer_links_title` - заголовок блока ссылок

## Безопасность
- Все внешние ссылки используют `rel="noopener noreferrer"` для безопасности
- `target="_blank"` для открытия в новой вкладке

## Особенности реализации
- **SEO:** Правильные семантические ссылки
- **Accessibility:** Alt тексты для иконок
- **Performance:** Статический компонент без тяжелых вычислений
- **Отслеживание:** Новая функциональность для улучшения UX пользователей
