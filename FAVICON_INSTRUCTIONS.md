# Инструкция по созданию favicon

Для полной оптимизации метаданных необходимо создать следующие файлы в директории `public/`:

## Необходимые файлы:

1. **favicon.ico** (16x16, 32x32, 48x48) - основной favicon
2. **favicon-16x16.png** - иконка 16x16 пикселей
3. **favicon-32x32.png** - иконка 32x32 пикселя
4. **apple-touch-icon.png** - иконка 180x180 пикселей для iOS устройств

## Рекомендации:

- Используйте существующий файл `/PaintBrush.svg` или `/logo.png` как основу
- Убедитесь, что иконки имеют прозрачный фон (для PNG файлов)
- Для apple-touch-icon используйте размер 180x180px без прозрачности
- Можно использовать онлайн-генераторы favicon, например:
  - https://realfavicongenerator.net/
  - https://favicon.io/

## Текущая конфигурация:

Сейчас используется `/PaintBrush.svg` как основной favicon. Для лучшей совместимости рекомендуется создать PNG версии.
