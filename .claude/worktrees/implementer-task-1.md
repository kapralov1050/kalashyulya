# Implementer Subagent - Task 1: Создать страницу успешной оплаты PaymentSuccess.vue

## Задача
Создать страницу `/shop/payment-success` которая отображает информацию об успешной оплате заказа по ID платежа от ЮKassa.

## Технологии
- Nuxt 3
- Vue 3
- TypeScript
- Firebase Realtime Database
- SCSS

## Требования к реализации

### Функциональные требования
1. Получить `paymentId` из route query параметров
2. Искать заказ в Firebase по `paymentId` используя `orderByChild('paymentId').equalTo(paymentId)`
3. Отобразить детали заказа (товар, цена, способ доставки, дата оплаты)
4. Показать номер для отслеживания (ID платежа) с предупреждением сохранить
5. Показать следующие шаги (свернуться с продавцом, отслеживание, контакты)
6. Показать контакт для связи (Telegram)
7. Обработать ошибки (заказ не найден, отсутствует paymentId, ошибка Firebase)
8. Добавить состояния загрузки (loading) и ошибки (error)
9. Использовать компонент AnimatedBlob для отображения галочки успеха

### UX требования
1. Использовать градиентный фон на странице
2. Карточки с тенью и скругленными углами
3. Кнопки действий (Вернуться в магазин, Страница отслеживания заказа)
4. Адаптивный дизайн для мобильных устройств
5. Темная тема (dark mode)
6. Анимация загрузки (spinner)
7. Сообщения об ошибках понятные на русском языке

### Технические требования
1. Использовать Nuxt composables (useRoute, useRouter, useFirebaseDatabase если нужно)
2. Типизация TypeScript для всех данных
3. SCSS для стилей с переменными цветов
4. Правильное использование Firebase Database API
5. Обработка ошибок Firebase с try-catch
6. Валидация входных данных (paymentId не должен быть пустым)

1. Создать тестовый заказ в Firebase с `paymentId`
2. Перейти по URL: `http://localhost:3000/shop/payment-success?paymentId=test_payment_id`
3. Проверить что:
